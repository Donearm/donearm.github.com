M2T

Published on 31 May 2019 10:54AM

In case you don’t know, magnet URIs are a standard scheme for identifying files by their content, and widespreadly used to distribute them by softwares supporting it (most of Bittorrent clients nowadays). Only legal files, of course.

Quite a lot of those clients support a “watch directory” in which you can drop a torrent file and start its download, immediately and automatically. For magnet URIs that is not possible though. This script I wrote transform such a URI into a file, saving it into those watch directories and allowing to automatically start downloading.

The repo is on Github. The script was written in pure Go.
